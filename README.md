# Homebrew Orca

Homebrew tap for [Orca](https://github.com/stablyai/orca) — an IDE for orchestrating AI coding agents.

## Install

```bash
brew tap stablyai/orca
brew install --cask orca
```

Or in one command:

```bash
brew install --cask stablyai/orca/orca
```

## Updates

Orca updates itself in place via its built-in updater. `brew upgrade` is a no-op unless you pass `--greedy`. `brew uninstall --cask orca` still works normally.

## About this repo

This tap is generated automatically from [stablyai/orca](https://github.com/stablyai/orca). The source of truth for `Casks/orca.rb` lives at [`Casks/orca.rb` in that repo](https://github.com/stablyai/orca/blob/main/Casks/orca.rb); do not hand-edit here.
